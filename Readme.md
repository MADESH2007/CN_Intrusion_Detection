# Simple Linear Regression — Interactive ML Lab

A browser-based interactive lab to learn and experiment with **Simple Linear Regression (SLR)**.

This project demonstrates how a regression model is built step-by-step, with real-time visualization and prediction capabilities.

---

## Overview

This application helps users understand linear regression by:

- Explaining core concepts and formulas  
- Walking through calculations step-by-step  
- Visualizing data points and regression line  
- Allowing custom dataset experimentation  

Designed for **students and beginners in machine learning**.

---

## Features

### Concept Explanation
- Mathematical model:
  ŷ = β₀ + β₁x
- Key parameters:
  - Intercept (β₀)
  - Slope (β₁)
  - SSE, MSE, RMSE
  - R² Score
  - Pearson Correlation

---

### Step-by-Step Execution
1. Load dataset  
2. Compute means  
3. Calculate deviations  
4. Compute slope (β₁)  
5. Compute intercept (β₀)  
6. Build final regression model  

---

### Interactive Lab
- Input custom X and Y values  
- Built-in dataset presets:
  - Study Hours vs Score  
  - Experience vs Salary  
  - House Area vs Price  
  - Temperature vs Sales  
- Step-by-step execution trace  

---

### Visualization
- Scatter plot  
- Regression line  
- Residual lines  
- Built using HTML Canvas  

---

### Prediction
- Enter any X value  
- Get predicted Y instantly

## Usage

1. Go to the **Lab section**  
2. Enter X and Y values (equal count)  
3. Click **START / NEXT STEP**  
4. Follow computation steps  
5. Use **Predict** for new values
